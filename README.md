# minimal-rothko-art

This project is a simple recreation of a Rothko-style abstract artwork using only HTML and CSS.

It explores how basic shapes, colors, blur, and positioning can be combined to create a digital version of abstract expressionist art.

🔧 Tech Used
HTML
CSS
🎨 What I Learned
CSS positioning and layout
Box shadows and blur effects
Transform and rotation for subtle realism
Creating visual depth with simple styling
🚀 Goal

To practice CSS by recreating modern abstract art using minimal code.
